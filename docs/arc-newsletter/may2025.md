# May 2025

### ARC API

1. **Caching Enhancements**: Updated the caching ARC API [@sourceloop/cache](https://www.npmjs.com/package/@sourceloop/cache) to make it more developer-friendly and efficient.

   **Key Updates**

   - New caching decorators to easily enable caching in any TypeScript class.
   - Improved cache invalidation via new decorators for granular and flexible control.
   - Repository mixins have been updated to use these decorators, promoting DRY principles.
   - Efficient tag-matching & deletion logic for faster cache management.

### ARC IaC

1. **EC2 Auto-scaling Module**: Our new [EC2 Auto Scaling Terraform module](https://github.com/sourcefuse/terraform-aws-arc-ec2-autoscale-group) offers production-ready support for dynamic EC2 workloads on AWS.

   **Key Features**

   - Built on the terraform-aws-arc-ec2-autoscale-group module.
   - Supports Launch Templates, IAM profiles, ALB/NLB integration, and CloudWatch alarms.
   - Dynamically scales based on real-time demand.
